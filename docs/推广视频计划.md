# 推广视频计划

## 定位

主线系列：**「找不到东西的崩溃瞬间」**。每期一个高频场景，从"找不到 XX"的情绪切入，落点永远是产品的 aha moment——问一句，AI 秒答位置 + 照片。

原则（2026-07-12 调整：不做实拍铺垫，全程录屏）：
- **第 0 秒就是产品画面**，场景用字幕压在录屏上表达，不实拍演场景。独立 App 的成熟打法是 UGC 式：hook（字幕）→ 问题 → 功能录屏 → 结果
- 打字问"退烧药放哪了"这个动作本身就是场景，输入框敲字的画面自带叙事，不需要铺垫
- 字幕短句（≤10 字一屏），大量用户静音刷，字幕是刚需；场景标题用**场景**（有情绪）不用类别："半夜找不到退烧药" ✓，"药品收纳" ✗
- 每期视频必须出现"问 → 答（位置+照片）"这一下，且在 10 秒内出现
- 单条制作成本压到最低（纯录屏+剪映字幕），多发多试——小红书推荐逻辑下小号起量靠数量和数据复盘
- 封面/标题统一模板，方便小红书做成合集：「找不到XX的第N年」

## 选题库（按优先级）

| 场景 | 崩溃瞬间 | 优先级 |
|------|---------|--------|
| 药品 | 半夜急用药翻遍抽屉 / 过期了才发现买过三份 | ⭐ 第一期 |
| 证件 | 出发前找不到护照/户口本/房产证 | ⭐ 第二期 |
| 充电线/数据线 | 明明买了一堆，用的时候一根都找不到 | 高 |
| 备用钥匙 | 锁在门外才想起不知道备用钥匙在哪 | 高 |
| 换季衣物 | 降温了，去年的秋裤收在哪个箱子 | 中（季节性，留到9-10月） |
| 孩子的东西 | 开学要交的材料/接种本在哪 | 中（开学季强） |
| 老人找东西 | 爸妈总问"我的XX呢"，帮他们记住全家 | 中（情感向，适合破圈） |
| 五金工具/小件 | 螺丝刀、电池、指甲刀永远在消失 | 低 |

## 第一期：找药 —— 录屏脚本

### 成片结构（目标 15-25 秒，全程录屏，字幕承担场景）

| 时间 | 画面（全是录屏） | 压屏字幕 |
|------|------|------|
| 0-3s | 录屏 B 段开头：助手页输入框逐字敲出「退烧药放哪了？」 | 「半夜找药，别翻抽屉了」 |
| 3-10s | AI 回答具体位置 + 照片，停 2-3 秒 | 「它记得比你清楚」（**aha moment**，原速） |
| 10-20s | 录屏 A 段倒叙：拍药品抽屉照片 → AI 识别出一串药名 → 确认保存 | 「提前拍一张照片就行」（识别过程 2x-4x 加速） |
| 结尾 2s | "我的家"页或 App 图标 | 「放哪了 · 家里的东西问它就行」固定片尾 |

> 敲字那 3 秒就是 hook：观众看到有人在问 AI"退烧药放哪了"，场景不言自明。字幕只补情绪，不解释画面。

### 录制前准备

1. **账号**：用有充足额度的账号（会员每日识别 50 次够用；免费号每天只有 1 次识别，别用）。账号里不要有真实敏感信息（证件照片等）。
2. **数据环境**：建议单独准备一个干净的演示家庭结构（比如只有"卧室"一个空间），或者直接用真实账号更有生活感——但要检查历史对话和空间里没有不想露出的内容。
3. **道具**：一个真实抽屉/药箱，摆 5-8 种药，品牌名朝上、光线好（识别准确率和展示效果都靠这个）。务必包含"布洛芬/泰诺林"这类退烧药，呼应 hook。
4. **手机设置**：开勿扰模式（防通知入镜）、电量充足、亮度拉高、关闭深色模式（App 是米白浅色方案）。iOS 控制中心开录屏。
5. **彩排 2-3 遍再正式录**：
   - AI 回答有随机性，先测几种问法（"退烧药在哪""布洛芬放哪了""家里有感冒药吗"），选回答最稳、最有展示效果（带照片、位置具体）的那句正式录
   - 大图偶尔识别超时（失败会退额度，但录屏得重来），正式录前先完整跑通一遍
   - 查询不扣识别次数，B 段可以放心多录几条选最好的

### 录屏 A 段：收录过程（先录这段，为 B 段准备数据）

1. 打开 App，进入**助手页**
2. 点拍照按钮，实拍药品抽屉（一张全景即可，多图反而拖节奏）
3. 照片进入输入框上方缩略图队列，输入一句说明：「卧室床头柜第一层抽屉的药」→ 发送
   - 这句说明很重要：既让 AI 归位准确，也向观众展示"可以说人话"
4. 等待 AI 识别，SSE 工作流步骤会逐条展示——**完整录下来**，剪辑时加速，这段是"AI 在干活"的可信度来源
5. 识别结果卡片（SuggestionCard）出现：药品清单 + 空间/位置 → 点**确认保存**
   - 如果识别结果有错漏，这条重录；演示视频里不要出现手动改错的过程
6. 切到**我的家** tab → 进入卧室 → 空间详情里看到"床头柜第一层抽屉"位置和药品列表
   - 这一屏是"存进去了"的视觉证据，扫一眼即可，成片里给 1-2 秒

### 录屏 B 段：查询（aha moment，成片里放在前面）

1. 回到助手页（如果 A 段刚录完，往上滑一下让画面从干净的输入框开始，或杀掉 App 重进）
2. 输入「退烧药放哪了？」→ 发送
   - **敲字要慢而清晰**：这 3 秒是成片的 hook，观众靠看你打的字理解场景，别用粘贴
   - 用**语义化问法**（"退烧药"而不是具体药名"布洛芬"），展示 AI 能理解模糊描述，这是和"备忘录记一下"的核心差异
3. AI 回答：具体到"卧室 → 床头柜第一层抽屉"，如果调了 view_photo 会附上当时拍的照片——**这一屏是整条视频的钱镜头**，停留 2-3 秒
4. 可以追问一句「家里还有什么感冒药」录备用素材，剪辑时看节奏取舍

### 备用素材（同场顺手录，后续视频用）

- 问「XX药过期了吗」（如果识别时存了有效期信息）
- 多图连拍识别（一次发 3 张照片）——留给「10 分钟数字化全家」辅线系列
- "我的家"页空间列表滑动浏览——留作片尾空镜

## 通用规范

- 时长：15-25 秒，最长不超过 30 秒
- 全程录屏，第 0 秒就是 App 画面，场景靠压屏字幕 + 打字内容表达（不实拍铺垫）
- 字幕用剪映自动识别 + 手动精修，每屏 ≤10 字；默认按静音观看设计
- 标题模板：「找不到XX的第N年」或「半夜/出门前 找不到XX」句式
- 封面：统一模板（场景大字 + App 截图小图），保持账号页整齐
- 每条视频发布后 48h 记录数据到进度表：播放、点赞、主页访问、App Store 跳转（如果能看到）
